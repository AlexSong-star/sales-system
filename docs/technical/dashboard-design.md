## 一、设计定位

### 1.1 界面定位
外贸销售团队的**数据指挥中心** — 简洁、专业、一目了然。

### 1.2 设计风格：**现代商务简约**
- 以数据为核心，不做花哨装饰
- 色彩克制，重点数据突出
- 信息密度适中，留白舒适

## 五、仪表盘组件设计

### 5.1 统计卡片（4个）

```
┌──────────────────┐
│  📬  今日询盘      │
│     23           │
│   ↑ 12% vs昨日   │
└──────────────────┘
```

每个卡片：
- 图标（渠道色）
- 指标名称
- 数字（大字体）
- 趋势标签（上涨绿/下跌红）

4个卡片排成一行：
1. **今日询盘**（总量）
2. **本周询盘**
3. **待回复**（警告色）
4. **本月成交**

### 5.2 趋势折线图

- X轴：日期（最近7天/30天）
- Y轴：询盘数量
- 3条线：邮件(蓝) / WhatsApp(绿) / 阿里站(橙)
- 底部图例，可点击切换显示/隐藏

### 5.3 渠道占比饼图

- 环形图，中间显示总量
- 右侧图例，显示数量和百分比
- 颜色对应各渠道

### 5.4 转化漏斗

```
[收到询盘 100]
    ↓
[已回复  75]   ← 第三步
    ↓
[跟进中  40]   ← 第二步
    ↓
[已成交  12]   ← 第一步（核心指标）
```

每步显示数量和转化率

### 5.5 待处理询盘提醒

红色边框高亮，显示：
- 超过24小时未回复的询盘
- 客户名 + 产品 + 超时时长
- 点击直接跳转详情

### 5.6 询盘列表（可展开详情）

| 渠道 | 客户 | 产品 | 时间 | 状态 | 操作 |
|------|------|------|------|------|------|

- 点击行任意位置展开详情（类似表格行展开）
- 展开后显示：原始询盘内容 + 所有回复记录
- 状态标签（待回复/已回复/跟进中/已成交/已无效）
- 仪表盘快捷区默认显示最近10条，全量在列表页查看

## 八、设计亮点（记忆点）

### 8.1 渠道色彩系统
三个渠道有固定色彩，用户一眼就能区分：
- 📧 邮件 → 蓝色 `#3B82F6`
- 💬 WhatsApp → 绿色 `#22C55E`
- 🛒 阿里站 → 橙色 `#F97316`

### 8.2 状态色彩语义
- 🔵 待回复 → 蓝色
- 🟢 已回复 → 绿色
- 🟡 跟进中 → 黄色/橙色
- ✅ 成交 → 深绿
- ⚫ 无效 → 灰色

### 8.3 超时警告
超过24小时未回复的询盘，列表行显示红色左边框，提醒优先处理

### 8.4 数字字体
统计数字使用 **Inter** 字体，加粗，字号比普通文字大50%

## 十、动效设计

- **页面切换**：淡入淡出，200ms ease
- **卡片hover**：轻微上浮 + 阴影加深，150ms
- **数字变化**：计数器动画，从0滚动到当前值
- **图表加载**：从左到右渐入，400ms
